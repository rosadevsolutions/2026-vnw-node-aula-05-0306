### Instalação e Inicialização do Projeto

Através do terminal de sua escolha:

1. **Download/Clonar o projeto para o seu dispositivo:**<br>
   [Repositório Remoto no Github](https://github.com/rosadevsolutions/2026-vnw-node-06-desafio-api-cadastro)<br>

2. **Acessar o projeto:**<br>
   Acesse a pasta raiz na qual o projeto foi clonado

3. **Instalar dependências de desenvolvimento:**<br>
   Digitar no terminal o comando `npm install`

4. **Inicializar/Rodar o servidor de desenvolvimento do projeto:**<br>
   Digitar no terminal o comando `npm run dev`

5. **Visualizar o projeto**:<br>
   Acessar o link gerado após inicializar o projeto. Copie o link e abra em seu navegador de preferência
